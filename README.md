# toxWhatsApp

Unofficial WhatsApp Desktop Client (currently only for OSX) based on electron

***

## was geht ...

* OSX-Client
* Auto-Update
* Badge im Dock bei neuen Nachrichten
* Mitteilung bei neuen Nachrichten
